插件

通过 extends 我们可以引入额外的配置和依赖，目前提供的插件有：

  • vue - 提供 vue 相关配置
  • sass - sass 配置
  • postcss - postcss-loader, 内置 cssnext 等
  • lint - 使用 ESLint
  • saladcss - 更好用的 CSS 预处理插件 postcss-salad

更多插件可以在 https://github.com/cookingjs 找到，同时欢迎你来开发新的插件。
